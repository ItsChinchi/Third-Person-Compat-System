# Third-Person Combat & Traversal — Unity
A third-person action game built in Unity from a GameDev.tv course, featuring fluid character movement, melee combat, and traversal mechanics.

## 🎮 Gameplay
https://drive.google.com/file/d/13DlJscDqQwMSsaRS3rxXI6MyuhzrSZhx/view?usp=sharing

## ✨ Features
- Third-person character controller with smooth locomotion
- Melee combat system with attack combos
- Enemy AI with state-based behaviour (idle, chase, attack)
- Targeting / lock-on system
- Dodge and block mechanics
- Health and stamina systems
- Ragdoll physics on enemy death
- Cinemachine-driven third-person camera

## 🧠 Concepts Applied
- State Machine pattern for player and enemy behaviour
- Root motion animation with Animator Controller
- Inverse Kinematics (IK) for foot placement
- NavMesh for enemy pathfinding
- Scriptable Objects for combat data
- Physics-based interaction and collision detection
- Camera control with Cinemachine Free Look

## 🛠 Built With
- Unity
- Cinemachine
- Unity NavMesh
- Animator / Root Motion
